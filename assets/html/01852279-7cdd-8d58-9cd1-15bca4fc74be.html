Si, se llega a calentar de la parte de las cuchillas y puede no cortar muy bien en algunos momentos luego de usarlo. Pero si se le da mantenimiento, limpiando la parte interna con el pequeño cepillo y se lubrican las cuchillas, la rasuradora funciona de maravilla. Se hace el rasurado en seco, no hay necesidad de espuma, crema ni debe estar la piel mojada ni muy húmeda. Y cabe decir que se puede cargar con el mismo cargador del teléfono (usb tipo c). Para el precio, es una muy buena rasuradora. Para todo tipo de vello, y su otra navaja para patillas es útil para recortar vello.
